Exercício de lógica de programação
---

Sabendo que a área quadrada é dada pela multiplicação dos lados, escreva um algoritmo que mostre a área quadrada de
um espaço qualquer.

Exemplo:

* A área de 3x9 é igual a 27
* 3 . 9 = 27



Linguagem C
---

Abaixo nosso código inicial escrito em C, utilize ele para resolver o problema.

```c
#include <stdio.h>
#include <assert.h>

//
// Função que retorna o valor da área
//
int area(int lado1, int lado2) {


}

//
// testes
//
int main (){
    assert(27 == area(3, 9));
    return 0;
}
```



### Solução na linguagem C

```c
#include <stdio.h>
#include <assert.h>

//
// Função que retorna o valor da área
//
int area(int lado1, int lado2) {
    return lado1 * lado2;
}

//
// testes
//
int main (){
    assert(27 == area(3, 9));
    return 0;
}
```
